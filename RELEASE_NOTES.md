Tibia Toolkit 0.4.0

- Added the complete Wheel of Destiny planner, with vocation selection, Promotion Points, code and URL import/export, slice controls, perks, gems, vessels, responsive layouts, and a guided tutorial.
- Reorganized Tools into Calculators and Hunting Tools. Imbuement and Skill calculators now share the Calculators category, while Party Hunt, Solo Hunt, and Find Party are grouped under Hunting Tools.
- Fixed the Wheel of Destiny floating point controls so each action is enabled only when the selected slice can perform it, matching the lower Selection controls.
