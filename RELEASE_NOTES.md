Tibia Toolkit 0.5.0

- Added Mini World Changes to the desktop app, with active changes for the selected world, a local catalog of 28 entries, offline details, maps, galleries, achievements, rewards, and internal links to items, NPCs, creatures, and bosses.
- Added Mini World Changes to the Tibia Toolkit website with the same local catalog and visual identity.
- Improved Tibia Mirror interaction stability, including right-click menus, mirror hit testing, profile/card synchronization, deletion responsiveness, and visual and sound alerts.
- Reduced startup load by moving non-critical Screen Vision, tutorial, and cache initialization out of the critical interface path.
- Added localized Mini World Changes content and interface updates in Portuguese, English, and German while preserving Tibia proper names.
- Improved resource-pack diagnostics and recovery for download, disk-space, integrity, and extraction failures.
